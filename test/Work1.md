# ออกแบบวิธีการทดสอบเซ็นเซอร์ KY-021 ให้แสดงผลที่ Serial Monitor

การทดสอบนี้เป็นการตรวจสอบการทำงานของโมดูลสวิตช์แม่เหล็กขนาดเล็ก (**KY-021 Mini Reed Switch Module**) ว่าสามารถตรวจจับสนามแม่เหล็กและส่งสัญญาณดิจิทัลได้อย่างถูกต้องหรือไม่ โดยจะทำการอ่านค่าสัญญาณแล้วส่งไปแสดงผลบนหน้าจอคอมพิวเตอร์ผ่านทาง Serial Monitor

---

## 1. อุปกรณ์ที่ใช้ในการทดสอบ

1. บอร์ดไมโครคอนโทรลเลอร์ (เช่น Arduino UNO R3) จำนวน 1 บอร์ด
2. โมดูลเซ็นเซอร์ **KY-021 Mini Reed Switch Module** จำนวน 1 ตัว 


3. แม่เหล็ก (สำหรับใช้ทดสอบการกระตุ้นเซ็นเซอร์) จำนวน 1 ก้อน
4. สายไฟจัมเปอร์ (Jumper Wires) แบบ เมีย-ผู้ (Female-to-Male) จำนวน 3 เส้น
5. สาย USB สำหรับต่อบอร์ด Arduino เข้ากับคอมพิวเตอร์

---

## 2. การต่อวงจร (Wiring Diagram)

เนื่องจากโมดูล KY-021 มีขาเชื่อมต่อทั้งหมด 3 ขา  (สัญญาณ, ไฟเลี้ยง, กราวด์) ให้ทำการต่อเข้ากับบอร์ด Arduino ดังนี้:

| ขาบนโมดูล KY-021 | ขาบนบอร์ด Arduino | หน้าที่ |
| --- | --- | --- |
| **S** (Signal) | **Digital Pin 2** | ส่งสัญญาณดิจิทัลเข้าบอร์ด 

 |
| **ขากลาง** (VCC / `_`) | **5V** หรือ **3.3V** | จ่ายไฟเลี้ยงให้โมดูล 

 |
| **-** (Ground) | **GND** | ต่อสายดิน/กราวด์ 

 |

> 
> หมายเหตุ: บนบอร์ดสกรีนสัญลักษณ์ `S` สำหรับสัญญาณ และมีตัวต้านทาน 10KΩ  แบบ Pull-up/Down อยู่บนโมดูลอยู่แล้ว
> 
>

## 3. ซอร์สโค้ดสำหรับทดสอบ (Arduino Code)

ให้คัดลอกโค้ดด้านล่างนี้ไปวางในโปรแกรม Arduino IDE จากนั้นทำการ Upload ลงบอร์ด:

```cpp
// กำหนดขา Digital Pin ที่ต่อกับขา Signal ของเซ็นเซอร์
const int reedPin = 2; 

void setup() {
  // เริ่มต้นการสื่อสาร Serial ที่ Baud Rate 9600
  Serial.begin(9600);
  
  // กำหนดให้ขาที่ต่อกับเซ็นเซอร์เป็นอินพุต (INPUT)
  pinMode(reedPin, INPUT);
  
  Serial.println("--- เริ่มต้นการทดสอบเซ็นเซอร์ KY-021 ---");
}

void loop() {
  // อ่านค่าสถานะดิจิทัลจากเซ็นเซอร์ (ได้ค่าเป็น 0 หรือ 1)
  int sensorState = digitalRead(reedPin);
  
  // ตรวจสอบเงื่อนไขและแสดงผลบน Serial Monitor
  if (sensorState == HIGH) {
    Serial.println("สถานะ:ปกติ (ON) - ไม่พบสนามแม่เหล็ก");
  } else {
    Serial.println("สถานะ: ตรวจพบสนามแม่เหล็ก! (Magnetic Field Detected)");
  }
  
  // หน่วงเวลา 500 มิลลิวินาที (0.5 วินาที) ก่อนอ่านค่าครั้งต่อไปเพื่อไม่ให้ข้อมูลวิ่งเร็วเกินไป
  delay(500);
}

```

## 4. ขั้นตอนการทดสอบและการดูผลลัพธ์

1. **ตรวจสอบความถูกต้อง:** ตรวจเช็คสายไฟว่าต่อสนิทและถูกขาตามตารางในข้อ 2
2. **อัปโหลดโค้ด:** เชื่อมต่อบอร์ด Arduino เข้าคอมพิวเตอร์และกดปุ่ม **Upload** ใน Arduino IDE
3. **เปิด Serial Monitor:** * ไปที่เมนู `Tools` > `Serial Monitor` (หรือกดปุ่ม `Ctrl + Shift + M`)
* ตั้งค่า Baud Rate ที่มุมขวาล่างของหน้าต่างให้ตรงกับโค้ด คือ **9600 baud**


4. **สังเกตผลลัพธ์:**
* 
**สภาวะปกติ:** สวิตช์ภายในจะต่อวงจรค้างไว้ (On)  หน้าจอจะขึ้นข้อความ *"สถานะ:ปกติ (ON) - ไม่พบสนามแม่เหล็ก"*


* **นำแม่เหล็กมาใกล้:** นำแม่เหล็กมาจ่อใกล้ๆ ตัวหลอดแก้วของโมดูล KY-021 หน้าจอจะเปลี่ยนเป็นข้อความ *"สถานะ: ตรวจพบสนามแม่เหล็ก!"* ทันที


5. **บันทึกผล:** หากข้อความเปลี่ยนตามการเอาแม่เหล็กเข้า-ออก แสดงว่าเซ็นเซอร์ทำงานปกติพร้อมใช้งาน
