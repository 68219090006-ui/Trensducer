# ออกแบบการนำ Sensor ไปประยุกต์ใช้งานเบื้องต้น

## หัวข้อโครงงาน
ระบบเปิดไฟอัตโนมัติด้วย PIR Motion Sensor และ Arduino

## แนวคิดของงาน
Sensor คืออุปกรณ์ที่ใช้ตรวจจับการเปลี่ยนแปลงของสภาพแวดล้อม เช่น แสง อุณหภูมิ ความชื้น ระยะทาง การเคลื่อนไหว หรือสนามแม่เหล็ก แล้วเปลี่ยนข้อมูลที่ตรวจจับได้เป็นสัญญาณไฟฟ้า เพื่อนำไปประมวลผลด้วยไมโครคอนโทรลเลอร์ เช่น Arduino

โครงงานนี้ออกแบบให้ใช้ **PIR Motion Sensor** ตรวจจับการเคลื่อนไหวของคน เมื่อมีคนเดินผ่านหรืออยู่ในบริเวณที่กำหนด Arduino จะอ่านค่าสัญญาณจากเซนเซอร์ แล้วสั่งให้ LED หรือหลอดไฟเปิดอัตโนมัติ หากไม่มีการเคลื่อนไหว ระบบจะปิดไฟ เพื่อช่วยประหยัดพลังงานและเพิ่มความสะดวกในการใช้งาน

## วัตถุประสงค์
1. เพื่อเรียนรู้การนำ Sensor ไปประยุกต์ใช้กับงานจริง
2. เพื่อทดลองอ่านค่าสัญญาณดิจิทัลจาก PIR Motion Sensor
3. เพื่อออกแบบระบบเปิด-ปิดไฟอัตโนมัติ
4. เพื่อฝึกเขียนโปรแกรมควบคุมอุปกรณ์ด้วย Arduino

## อุปกรณ์ที่ใช้
| ลำดับ | อุปกรณ์ | จำนวน |
| --- | --- | --- |
| 1 | Arduino UNO หรือบอร์ดไมโครคอนโทรลเลอร์ | 1 บอร์ด |
| 2 | PIR Motion Sensor | 1 ตัว |
| 3 | LED | 1 ดวง |
| 4 | ตัวต้านทาน 220 โอห์ม | 1 ตัว |
| 5 | สาย Jumper | ตามต้องการ |
| 6 | สาย USB สำหรับเชื่อมต่อ Arduino กับคอมพิวเตอร์ | 1 เส้น |

> หากต้องการควบคุมหลอดไฟจริง สามารถใช้ Relay Module แทน LED ได้ แต่ต้องระวังเรื่องไฟบ้านและความปลอดภัยเป็นพิเศษ

## หลักการทำงาน
1. PIR Motion Sensor ตรวจจับการเคลื่อนไหวของคนจากการเปลี่ยนแปลงของรังสีอินฟราเรด
2. เมื่อพบการเคลื่อนไหว เซนเซอร์จะส่งสัญญาณดิจิทัลไปยัง Arduino
3. Arduino ตรวจสอบค่าสัญญาณจากขา Digital
4. ถ้าตรวจพบการเคลื่อนไหว Arduino จะสั่งให้ LED เปิด
5. ถ้าไม่พบการเคลื่อนไหว Arduino จะสั่งให้ LED ปิด

## การต่อวงจร
| ขา PIR Sensor | ขา Arduino UNO | หน้าที่ |
| --- | --- | --- |
| `VCC` | `5V` | จ่ายไฟเลี้ยงให้เซนเซอร์ |
| `GND` | `GND` | ต่อกราวด์ |
| `OUT` | `D2` | ส่งสัญญาณดิจิทัลเข้า Arduino |

| ขา LED | ขา Arduino UNO | หน้าที่ |
| --- | --- | --- |
| ขายาวของ LED | `D13` ผ่านตัวต้านทาน 220 โอห์ม | รับสัญญาณควบคุม |
| ขาสั้นของ LED | `GND` | ต่อกราวด์ |

## แผนผังการทำงานของระบบ
```text
คนเดินผ่านบริเวณที่ตรวจจับ
        ↓
PIR Sensor ตรวจจับการเคลื่อนไหว
        ↓
ส่งสัญญาณ HIGH หรือ LOW ไปยัง Arduino
        ↓
Arduino ประมวลผลเงื่อนไข
        ↓
เปิดหรือปิด LED อัตโนมัติ
```

## ตัวอย่างโค้ด Arduino
```cpp
const int pirPin = 2;
const int ledPin = 13;

void setup() {
  pinMode(pirPin, INPUT);
  pinMode(ledPin, OUTPUT);
  Serial.begin(9600);
  Serial.println("เริ่มต้นระบบเปิดไฟอัตโนมัติ");
}

void loop() {
  int motionState = digitalRead(pirPin);

  if (motionState == HIGH) {
    digitalWrite(ledPin, HIGH);
    Serial.println("ตรวจพบการเคลื่อนไหว: เปิดไฟ");
  } else {
    digitalWrite(ledPin, LOW);
    Serial.println("ไม่พบการเคลื่อนไหว: ปิดไฟ");
  }

  delay(500);
}
```

## ขั้นตอนการทดสอบ
1. ต่อวงจรตามตารางการต่อวงจร
2. เปิดโปรแกรม Arduino IDE
3. คัดลอกโค้ดตัวอย่างลงในโปรแกรม
4. Upload โค้ดลงบอร์ด Arduino
5. เปิด Serial Monitor และตั้งค่า Baud Rate เป็น `9600`
6. รอให้ PIR Sensor เริ่มทำงานประมาณ 30-60 วินาที
7. ทดลองเดินผ่านหน้าเซนเซอร์
8. สังเกตสถานะ LED และข้อความบน Serial Monitor

## ตัวอย่างการนำไปประยุกต์ใช้งาน
- เปิดไฟทางเดินอัตโนมัติเมื่อมีคนเดินผ่าน
- ระบบไฟห้องน้ำหรือห้องเก็บของอัตโนมัติ
- ระบบแจ้งเตือนเมื่อมีคนเข้าพื้นที่
- ระบบรักษาความปลอดภัยเบื้องต้น
- ระบบประหยัดพลังงานภายในบ้านหรืออาคาร

## ข้อควรระวัง
- ตรวจสอบขา `VCC`, `GND` และ `OUT` ให้ถูกต้องก่อนจ่ายไฟ
- ควรรอให้ PIR Sensor พร้อมทำงานก่อนเริ่มทดสอบ
- หลีกเลี่ยงการติดตั้งใกล้แหล่งความร้อน เช่น แสงแดดจัดหรือเครื่องทำความร้อน
- หากใช้ Relay ควบคุมหลอดไฟบ้าน ต้องระวังไฟฟ้าแรงสูง และควรให้ผู้มีความรู้ช่วยตรวจสอบ
- ปรับตำแหน่งและมุมของเซนเซอร์ให้เหมาะสมกับพื้นที่ที่ต้องการตรวจจับ

## สรุป
การนำ Sensor ไปประยุกต์ใช้งานเบื้องต้นสามารถเริ่มจากระบบเปิดไฟอัตโนมัติด้วย PIR Motion Sensor และ Arduino โครงงานนี้ช่วยให้เข้าใจการตรวจจับการเคลื่อนไหว การอ่านค่าสัญญาณดิจิทัล และการนำข้อมูลจากเซนเซอร์ไปควบคุมอุปกรณ์จริง ซึ่งเป็นพื้นฐานสำคัญของระบบอัตโนมัติ บ้านอัจฉริยะ และงาน IoT
