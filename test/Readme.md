## เทคนิคคอมพิวเตอร์

## LBA03 DHT11

### ข้อมูลพื้นฐานของ Sensor DHT11

DHT11 คือเซนเซอร์วัดอุณหภูมิและความชื้นแบบดิจิทัลที่นิยมใช้ในงานทดลองและงานพื้นฐานด้านไมโครคอนโทรลเลอร์ เช่น Arduino และ ESP32 เพราะต่อวงจรง่าย ราคาไม่แพง และอ่านค่าผ่านขา Digital ได้โดยตรง

### คุณสมบัติสำคัญ

- วัดอุณหภูมิได้ประมาณ 0 ถึง 50 องศาเซลเซียส
- วัดความชื้นได้ประมาณ 20% ถึง 90% RH
- ความละเอียดการวัดอุณหภูมิ 1 องศาเซลเซียส
- ความละเอียดการวัดความชื้น 1% RH
- ใช้ไฟเลี้ยงประมาณ 3.3V ถึง 5V
- ส่งข้อมูลแบบสัญญาณดิจิทัลผ่านสายข้อมูลเส้นเดียว

### ขาใช้งานของ DHT11

โดยทั่วไป DHT11 แบบ 4 ขา จะมีขาใช้งานดังนี้

1. VCC สำหรับจ่ายไฟ
2. DATA สำหรับรับ-ส่งข้อมูล
3. NC หรือไม่ใช้งานในบางรุ่น
4. GND สำหรับกราวด์

### หลักการทำงาน

เมื่อไมโครคอนโทรลเลอร์ส่งสัญญาณเริ่มต้นไปยัง DHT11 เซนเซอร์จะตอบกลับข้อมูลอุณหภูมิและความชื้นในรูปแบบดิจิทัล จากนั้นบอร์ดจะนำค่าที่ได้ไปแสดงผลหรือประมวลผลต่อ

### การใช้งานเบื้องต้น

สามารถนำ DHT11 ไปใช้ในงานต่าง ๆ เช่น

- วัดอุณหภูมิและความชื้นในห้องเรียน
- ระบบแจ้งเตือนเมื่ออากาศร้อนหรือชื้นเกินไป
- โครงงาน IoT ที่ต้องการเก็บข้อมูลสภาพอากาศ
- ระบบควบคุมพัดลมหรืออุปกรณ์ระบายอากาศ

### ข้อควรรู้

- DHT11 เหมาะกับงานพื้นฐาน ไม่เหมาะกับงานที่ต้องการความแม่นยำสูง
- หากต้องการความละเอียดและช่วงวัดที่กว้างกว่า ควรพิจารณาใช้ DHT22
- ควรเว้นระยะเวลาอ่านค่าตามที่ไลบรารีกำหนด เพื่อป้องกันการอ่านค่าผิดพลาด

### สรุป

DHT11 เป็นเซนเซอร์ที่ใช้งานง่าย เหมาะสำหรับการเรียนรู้การอ่านค่าจากเซนเซอร์ดิจิทัล และเป็นจุดเริ่มต้นที่ดีสำหรับการทำโครงงานเกี่ยวกับอุณหภูมิและความชื้น
