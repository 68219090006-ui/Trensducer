# สรุปความรู้พื้นฐานเกี่ยวกับ Sensor

## Sensor คืออะไร
Sensor คืออุปกรณ์ที่ใช้ตรวจจับการเปลี่ยนแปลงของสภาพแวดล้อม เช่น อุณหภูมิ แสง ความชื้น ระยะทาง ความดัน การเคลื่อนไหว หรือแรงสั่นสะเทือน แล้วแปลงข้อมูลนั้นเป็นสัญญาณไฟฟ้าเพื่อนำไปประมวลผลต่อ

## หลักการทำงานเบื้องต้น
1. Sensor รับค่าจากสิ่งแวดล้อม
2. แปลงค่าที่ตรวจจับได้เป็นสัญญาณไฟฟ้า
3. ส่งสัญญาณไปยังไมโครคอนโทรลเลอร์หรือวงจรควบคุม
4. ระบบนำข้อมูลไปแสดงผล ตัดสินใจ หรือสั่งงานอุปกรณ์อื่น

## ประเภทของ Sensor ที่พบบ่อย
- Sensor วัดอุณหภูมิ เช่น DHT11, DHT22, LM35
- Sensor วัดแสง เช่น LDR, Photoresistor
- Sensor วัดระยะทาง เช่น Ultrasonic, Infrared
- Sensor ตรวจจับการเคลื่อนไหว เช่น PIR
- Sensor วัดความชื้น เช่น Soil Moisture Sensor
- Sensor ตรวจจับแรงกดหรือแรงสั่นสะเทือน เช่น Force Sensor, Vibration Sensor

## สัญญาณเอาต์พุตของ Sensor
- Analog Output: ให้ค่าเป็นช่วงต่อเนื่อง ต้องอ่านผ่านขา Analog
- Digital Output: ให้ค่า 0 หรือ 1 ใช้งานง่ายกับขา Digital
- I2C / SPI / UART: ใช้สื่อสารแบบดิจิทัลกับเซนเซอร์ที่ซับซ้อนกว่า

## ค่าที่ควรรู้เวลาเลือกใช้
- ช่วงการวัด: Sensor วัดได้ต่ำสุดถึงสูงสุดเท่าไร
- ความแม่นยำ: ค่าที่อ่านได้ใกล้ค่าจริงแค่ไหน
- ความละเอียด: แยกความต่างของค่าเล็ก ๆ ได้ดีเพียงใด
- เวลาตอบสนอง: Sensor ใช้เวลานานแค่ไหนในการตรวจจับ
- แรงดันไฟเลี้ยง: ใช้ไฟกี่โวลต์
- ระยะทำงาน: โดยเฉพาะ Sensor วัดระยะหรือตรวจจับการเคลื่อนไหว

## การต่อใช้งานเบื้องต้น
- ตรวจสอบขา VCC, GND และ OUT ให้ถูกต้อง
- ใช้แรงดันไฟให้ตรงกับสเปกของอุปกรณ์
- ถ้าเป็น Analog ให้อ่านค่าผ่านขา Analog ของบอร์ด
- ถ้าเป็น Digital ให้ดูว่าเอาต์พุตเป็น HIGH/LOW แบบไหน
- หากเป็นโมดูลสำเร็จรูป ให้อ่านคู่มือก่อนต่อทุกครั้ง

## ปัจจัยที่ทำให้ค่าเพี้ยน
- สายสัญญาณยาวเกินไปหรือคุณภาพสายไม่ดี
- แหล่งจ่ายไฟไม่เสถียร
- ตำแหน่งติดตั้งไม่เหมาะสม
- สัญญาณรบกวนจากอุปกรณ์ใกล้เคียง
- ต้องคาลิเบรตหรือปรับค่า offset ก่อนใช้งานจริง

## ตัวอย่างการใช้งาน
- วัดอุณหภูมิห้องแล้วแสดงผลบนหน้าจอ
- ตรวจจับคนเข้าออกด้วย PIR Sensor
- วัดระดับน้ำหรือความชื้นในดินสำหรับงานเกษตร
- ตรวจจับระยะห่างเพื่อหลบสิ่งกีดขวางของหุ่นยนต์

## สรุป
Sensor คือส่วนสำคัญของระบบอัตโนมัติและ IoT เพราะทำหน้าที่รับข้อมูลจากโลกจริงแล้วส่งต่อให้ไมโครคอนโทรลเลอร์ประมวลผล การเลือกใช้ Sensor ควรดูชนิดสัญญาณ ช่วงการวัด ความแม่นยำ และแรงดันไฟให้เหมาะกับงาน
