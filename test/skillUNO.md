# โครงสร้างบอร์ด Arduino UNO R3 และขาใช้งาน

## ภาพรวมของ UNO R3
Arduino UNO R3 เป็นบอร์ดไมโครคอนโทรลเลอร์ที่นิยมใช้มากในงานเรียนรู้และงานต้นแบบ เพราะใช้งานง่าย ต่อวงจรง่าย และมีเอกสารประกอบจำนวนมาก บอร์ดรุ่นนี้ใช้ชิป ATmega328P เป็นหลัก จึงเหมาะกับการเริ่มต้นเขียนโปรแกรมควบคุมอุปกรณ์อิเล็กทรอนิกส์

## โครงสร้างของบอร์ด
ส่วนประกอบสำคัญบนบอร์ด UNO R3 ได้แก่:
- พอร์ต USB สำหรับอัปโหลดโปรแกรมและจ่ายไฟ
- แจ็กไฟเลี้ยงภายนอก (DC Jack)
- ขา Digital I/O สำหรับรับและส่งสัญญาณ
- ขา Analog Input สำหรับอ่านค่าจากเซนเซอร์
- ขา Power เช่น 5V, 3.3V, GND และ Vin
- ปุ่ม Reset สำหรับเริ่มการทำงานใหม่
- ไฟ LED บนบอร์ดสำหรับแสดงสถานะ

## ขาใช้งานหลัก
### 1. ขา Digital I/O
ใช้รับหรือส่งสัญญาณแบบ HIGH/LOW
- มีทั้งหมด 14 ขา คือ D0 ถึง D13
- ขาบางส่วนใช้สำหรับงานพิเศษ เช่น สื่อสารอนุกรมหรือ PWM
- เหมาะกับ LED, ปุ่มกด, รีเลย์ และโมดูลดิจิทัลต่าง ๆ

### 2. ขา Analog Input
ใช้สำหรับอ่านค่าที่เปลี่ยนแปลงต่อเนื่อง
- มีทั้งหมด 6 ขา คือ A0 ถึง A5
- ใช้กับเซนเซอร์วัดแสง, โพเทนชิออมิเตอร์, เซนเซอร์วัดความชื้น

### 3. ขา PWM
บางขาดิจิทัลสามารถใช้สร้างสัญญาณ PWM ได้
- ใช้ปรับความสว่างของ LED
- ใช้ควบคุมความเร็วมอเตอร์
- ขาที่ใช้ PWM ได้มักระบุด้วยเครื่องหมาย `~`

### 4. ขา Serial Communication
ใช้สื่อสารแบบอนุกรมกับคอมพิวเตอร์หรืออุปกรณ์ภายนอก
- D0 คือ RX
- D1 คือ TX
- ใช้กับการส่งข้อมูล Debug หรือเชื่อมต่อโมดูลสื่อสาร

### 5. ขา I2C และ SPI
ใช้เชื่อมต่อเซนเซอร์และอุปกรณ์สื่อสารแบบดิจิทัล
- I2C ใช้ขา A4 (SDA) และ A5 (SCL)
- SPI ใช้ขา D10 (SS), D11 (MOSI), D12 (MISO), D13 (SCK)

## ขาไฟเลี้ยง
- 5V: จ่ายไฟให้โมดูลที่รองรับ 5 โวลต์
- 3.3V: ใช้กับอุปกรณ์ที่ต้องการแรงดันต่ำ
- GND: ขากราวด์ร่วมของวงจร
- Vin: ใช้รับไฟจากแหล่งจ่ายภายนอก
- IOREF: อ้างอิงแรงดันของขา I/O

## ตัวอย่างการใช้งานขา
- ขา Digital: ต่อปุ่มกดหรือ LED
- ขา Analog: ต่อเซนเซอร์วัดแสงหรือโพเทนชิออมิเตอร์
- ขา PWM: หรี่ไฟ LED
- ขา I2C: ต่อจอ OLED หรือเซนเซอร์ที่รองรับ I2C
- ขา Serial: ต่อโมดูลสื่อสารหรือดูข้อมูลผ่าน Serial Monitor

## ข้อควรระวัง
- ตรวจสอบแรงดันไฟก่อนต่ออุปกรณ์ทุกครั้ง
- อย่าต่ออุปกรณ์ที่กินกระแสสูงโดยตรงจากขา I/O
- ระวังการต่อขา D0 และ D1 เพราะใช้กับการสื่อสาร Serial
- ตรวจสอบผังขาก่อนใช้งานจริงเพื่อป้องกันต่อสายผิด

## สรุป
UNO R3 เป็นบอร์ดที่เหมาะสำหรับผู้เริ่มต้นและงานทดลองอิเล็กทรอนิกส์ ขาใช้งานหลักประกอบด้วย Digital, Analog, PWM, Serial, I2C และ SPI การเข้าใจหน้าที่ของขาแต่ละส่วนจะช่วยให้ต่อวงจรได้ถูกต้องและปลอดภัยมากขึ้น
