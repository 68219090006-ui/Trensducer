# โครงสร้างบอร์ด Arduino UNO R3 และขาใช้งาน

## ภาพรวมของ UNO R3
Arduino UNO R3 เป็นบอร์ดไมโครคอนโทรลเลอร์ที่นิยมใช้มากในงานเรียนรู้และงานต้นแบบ เพราะใช้งานง่าย ต่อวงจรง่าย และมีเอกสารประกอบจำนวนมาก บอร์ดรุ่นนี้ใช้ชิป ATmega328P เป็นหลัก จึงเหมาะกับการเริ่มต้นเขียนโปรแกรมควบคุมอุปกรณ์อิเล็กทรอนิกส์

## โครงสร้างของบอร์ด
ส่วนประกอบสำคัญบนบอร์ด UNO R3 ได้แก่:
- พอร์ต USB สำหรับอัปโหลดโปรแกรมและจ่ายไฟ
- แจ็กไฟเลี้ยงภายนอก (DC Jack)
- ขา Digital I/O สำหรับรับและส่งสัญญาณ
- ขา Analog Input สำหรับอ่านค่าจากเซนเซอร์
- ขา Power เช่น 5V, 3.3V, GND และ Vin
- ปุ่ม Reset สำหรับเริ่มการทำงานใหม่
- ไฟ LED บนบอร์ดสำหรับแสดงสถานะ

## ขาใช้งานหลัก
### 1. ขา Digital I/O
ใช้รับหรือส่งสัญญาณแบบ HIGH/LOW
- มีทั้งหมด 14 ขา คือ D0 ถึง D13
- ขาบางส่วนใช้สำหรับงานพิเศษ เช่น สื่อสารอนุกรมหรือ PWM
- เหมาะกับ LED, ปุ่มกด, รีเลย์ และโมดูลดิจิทัลต่าง ๆ

### 2. ขา Analog Input
ใช้สำหรับอ่านค่าที่เปลี่ยนแปลงต่อเนื่อง
- มีทั้งหมด 6 ขา คือ A0 ถึง A5
- ใช้กับเซนเซอร์วัดแสง, โพเทนชิออมิเตอร์, เซนเซอร์วัดความชื้น

### 3. ขา PWM
บางขาดิจิทัลสามารถใช้สร้างสัญญาณ PWM ได้
- ใช้ปรับความสว่างของ LED
- ใช้ควบคุมความเร็วมอเตอร์
- ขาที่ใช้ PWM ได้มักระบุด้วยเครื่องหมาย `~`

### 4. ขา Serial Communication
ใช้สื่อสารแบบอนุกรมกับคอมพิวเตอร์หรืออุปกรณ์ภายนอก
- D0 คือ RX
- D1 คือ TX
- ใช้กับการส่งข้อมูล Debug หรือเชื่อมต่อโมดูลสื่อสาร

### 5. ขา I2C และ SPI
ใช้เชื่อมต่อเซนเซอร์และอุปกรณ์สื่อสารแบบดิจิทัล
- I2C ใช้ขา A4 (SDA) และ A5 (SCL)
- SPI ใช้ขา D10 (SS), D11 (MOSI), D12 (MISO), D13 (SCK)

## ขาไฟเลี้ยง
- 5V: จ่ายไฟให้โมดูลที่รองรับ 5 โวลต์
- 3.3V: ใช้กับอุปกรณ์ที่ต้องการแรงดันต่ำ
- GND: ขากราวด์ร่วมของวงจร
- Vin: ใช้รับไฟจากแหล่งจ่ายภายนอก
- IOREF: อ้างอิงแรงดันของขา I/O

## ตัวอย่างการใช้งานขา
- ขา Digital: ต่อปุ่มกดหรือ LED
- ขา Analog: ต่อเซนเซอร์วัดแสงหรือโพเทนชิออมิเตอร์
- ขา PWM: หรี่ไฟ LED
- ขา I2C: ต่อจอ OLED หรือเซนเซอร์ที่รองรับ I2C
- ขา Serial: ต่อโมดูลสื่อสารหรือดูข้อมูลผ่าน Serial Monitor

## ข้อควรระวัง
- ตรวจสอบแรงดันไฟก่อนต่ออุปกรณ์ทุกครั้ง
- อย่าต่ออุปกรณ์ที่กินกระแสสูงโดยตรงจากขา I/O
- ระวังการต่อขา D0 และ D1 เพราะใช้กับการสื่อสาร Serial
- ตรวจสอบผังขาก่อนใช้งานจริงเพื่อป้องกันต่อสายผิด

## สรุป
UNO R3 เป็นบอร์ดที่เหมาะสำหรับผู้เริ่มต้นและงานทดลองอิเล็กทรอนิกส์ ขาใช้งานหลักประกอบด้วย Digital, Analog, PWM, Serial, I2C และ SPI การเข้าใจหน้าที่ของขาแต่ละส่วนจะช่วยให้ต่อวงจรได้ถูกต้องและปลอดภัยมากขึ้น

# โครงสร้างบอร์ด UNO R4 และขาใช้งาน

## ภาพรวมของ UNO R4
UNO R4 คือบอร์ดไมโครคอนโทรลเลอร์ตระกูล Arduino UNO รุ่นใหม่ที่ยังคงรูปแบบการใช้งานใกล้เคียงรุ่นเดิม แต่เพิ่มประสิทธิภาพและความสามารถให้เหมาะกับงานเรียนรู้ งานทดลอง และงานควบคุมอุปกรณ์อิเล็กทรอนิกส์

UNO R4 ที่พบได้บ่อยมี 2 รุ่น:
- UNO R4 Minima: รุ่นพื้นฐานสำหรับงานทั่วไป
- UNO R4 WiFi: รุ่นที่เพิ่มการเชื่อมต่อ Wi-Fi และความสามารถด้านการสื่อสาร

## สรุปโครงสร้างหลัก
| ส่วนประกอบ | หน้าที่ |
|---|---|
| ขาไฟเลี้ยง | จ่ายไฟให้บอร์ดและอุปกรณ์ภายนอก |
| ขา Digital I/O | รับและส่งสัญญาณแบบ HIGH/LOW |
| ขา Analog Input | อ่านค่าที่เปลี่ยนแปลงต่อเนื่อง |
| ขา PWM | ควบคุมความสว่าง ความเร็ว หรือกำลังสัญญาณ |
| ขาสื่อสาร | เชื่อมต่ออุปกรณ์ผ่าน UART, I2C หรือ SPI |
| ปุ่มและไฟแสดงสถานะ | ใช้ควบคุมและตรวจสอบการทำงานของบอร์ด |

## ขาใช้งานที่ควรรู้
### 1. ขา Digital I/O
ใช้รับหรือส่งสัญญาณดิจิทัลแบบ HIGH/LOW
- รุ่น UNO R4 Minima มีขาดิจิทัลใช้งานทั่วไป 14 ขา
- เหมาะกับ LED, ปุ่มกด, รีเลย์ และโมดูลดิจิทัลอื่น ๆ

### 2. ขา Analog Input
ใช้สำหรับอ่านค่าจากเซนเซอร์ที่มีสัญญาณเปลี่ยนแปลงต่อเนื่อง
- รุ่น UNO R4 Minima มีขาอนาล็อก 6 ขา
- เหมาะกับ LDR, Potentiometer และเซนเซอร์วัดความชื้น

### 3. ขา PWM
บางขาดิจิทัลสามารถสร้างสัญญาณ PWM เพื่อปรับระดับเอาต์พุตได้
- ใช้หรี่ความสว่างของ LED
- ใช้ควบคุมความเร็วมอเตอร์
- ใช้สร้างสัญญาณควบคุมอุปกรณ์บางประเภท

### 4. ขา Serial Communication
ใช้สื่อสารกับคอมพิวเตอร์หรืออุปกรณ์ภายนอกผ่านสัญญาณอนุกรม
- ใช้กับโมดูลบลูทูธ, GPS หรืออุปกรณ์สื่อสารอื่น
- เหมาะกับการส่งข้อมูลเข้าออกแบบต่อเนื่อง

### 5. ขา I2C และ SPI
ใช้เชื่อมต่อเซนเซอร์และโมดูลที่ต้องการการสื่อสารแบบดิจิทัล
- I2C เหมาะกับอุปกรณ์หลายตัวบนบัสเดียว
- SPI เหมาะกับงานที่ต้องการความเร็วในการสื่อสารสูง

## ขาไฟเลี้ยง
- 5V: จ่ายไฟให้โมดูลหรืออุปกรณ์ที่รองรับ 5 โวลต์
- 3.3V: ใช้กับอุปกรณ์ที่ต้องการแรงดันต่ำกว่า
- GND: ขากราวด์ร่วมของวงจร
- Vin: ใช้รับไฟจากแหล่งจ่ายภายนอกตามสเปกของบอร์ด

## ตัวอย่างการใช้งาน
- ขา Digital: ต่อปุ่มกดหรือ LED
- ขา Analog: ต่อเซนเซอร์วัดแสงหรือโพเทนชิออมิเตอร์
- ขา PWM: หรี่ความสว่างของ LED
- ขา I2C: ต่อจอ OLED หรือเซนเซอร์หลายชนิด
- ขา Serial: ต่อโมดูลสื่อสารภายนอก

## การแมปขาบน UNO R4 Minima
- D4 ใช้เป็น CANTX0 สำหรับสัญญาณ CAN TX
- D5 ใช้เป็น CANRX0 สำหรับสัญญาณ CAN RX

## ความสามารถของ UNO R4 Minima
- ความถี่การทำงาน 48 MHz
- SRAM 32 kB
- Flash 256 kB
- EEPROM 8 kB

## ข้อควรระวัง
- ตรวจสอบแรงดันไฟก่อนต่ออุปกรณ์ทุกครั้ง
- อย่าต่ออุปกรณ์ที่กินกระแสเกินความสามารถของบอร์ดโดยตรง
- ตรวจสอบว่าขาใช้งานรองรับแรงดัน 5V หรือไม่
- ศึกษาผังขาของรุ่นที่ใช้อยู่จริง เพราะ UNO R4 Minima และ UNO R4 WiFi อาจมีรายละเอียดต่างกัน

## สรุป
UNO R4 เป็นบอร์ดที่ใช้งานง่ายและเหมาะกับงานทดลองอิเล็กทรอนิกส์ โดยขาใช้งานหลักประกอบด้วย Digital, Analog, PWM, Serial, I2C และ SPI การเข้าใจหน้าที่ของขาแต่ละประเภทจะช่วยให้ต่อวงจรได้ถูกต้อง ปลอดภัย และนำไปประยุกต์ใช้ได้หลากหลาย
