# KY-021 Magnetic Reed Switch Sensor

## ข้อมูลทั่วไป
KY-021 คือโมดูลเซนเซอร์สวิตช์แม่เหล็กแบบ Reed Switch ใช้ตรวจจับสนามแม่เหล็กหรือการเข้าใกล้ของแม่เหล็ก เมื่อมีแม่เหล็กมาอยู่ใกล้ หน้าสัมผัสภายในหลอด Reed จะเปลี่ยนสถานะ ทำให้ขา `S` ส่งสัญญาณดิจิทัลออกมาเป็น `HIGH` หรือ `LOW`

โมดูลนี้เหมาะกับงานตรวจจับการเปิด-ปิดประตู หน้าต่าง ฝาเครื่อง กล่องเก็บของ หรือใช้เป็นสวิตช์แบบไม่ต้องสัมผัสในงาน Arduino และไมโครคอนโทรลเลอร์

## หลักการทำงาน
ภายในเซนเซอร์มี Reed Switch ซึ่งเป็นสวิตช์ขนาดเล็กที่ตอบสนองต่อสนามแม่เหล็ก

1. เมื่อไม่มีแม่เหล็กอยู่ใกล้ หน้าสัมผัสภายในจะอยู่ในสถานะปกติ
2. เมื่อมีแม่เหล็กเข้าใกล้ สนามแม่เหล็กจะทำให้หน้าสัมผัสเปลี่ยนสถานะ
3. ไมโครคอนโทรลเลอร์สามารถอ่านค่าจากขา `S` เพื่อตรวจสอบว่ามีแม่เหล็กอยู่ใกล้หรือไม่

## ขาเชื่อมต่อ

| ขา | รายละเอียด |
| --- | --- |
| `S` | ขาสัญญาณดิจิทัล ส่งค่า `HIGH` หรือ `LOW` |
| `+` | ไฟเลี้ยง เช่น `5V` หรือ `3.3V` ตามบอร์ดที่ใช้ |
| `-` | กราวด์ `GND` |

## อุปกรณ์ที่ใช้
- Arduino UNO หรือบอร์ดไมโครคอนโทรลเลอร์อื่น
- โมดูล KY-021 Magnetic Reed Switch
- แม่เหล็ก
- สาย Jumper
- LED หรือใช้ไฟ `LED_BUILTIN` บนบอร์ด Arduino

## การต่อวงจรกับ Arduino UNO

| KY-021 | Arduino UNO |
| --- | --- |
| `S` | `D2` |
| `+` | `5V` |
| `-` | `GND` |

## ตัวอย่างโค้ด Arduino

```cpp
const int sensorPin = 2;
const int ledPin = 13;

void setup() {
  pinMode(sensorPin, INPUT);
  pinMode(ledPin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int sensorValue = digitalRead(sensorPin);

  if (sensorValue == HIGH) {
    digitalWrite(ledPin, HIGH);
    Serial.println("Magnet detected");
  } else {
    digitalWrite(ledPin, LOW);
    Serial.println("No magnet");
  }

  delay(200);
}
```

> หมายเหตุ: บางโมดูลอาจให้ค่ากลับกัน เช่น มีแม่เหล็กแล้วอ่านได้ `LOW` ขึ้นอยู่กับวงจรบนโมดูล หากผลลัพธ์สลับกันให้เปลี่ยนเงื่อนไข `HIGH` และ `LOW` ในโค้ด

## การนำไปใช้งาน
- ตรวจจับประตูหรือหน้าต่างเปิด-ปิด
- ระบบกันขโมยขนาดเล็ก
- ตรวจจับตำแหน่งฝาปิดหรือกล่อง
- ใช้เป็นสวิตช์แม่เหล็กแบบไม่ต้องสัมผัส
- ใช้ในงานนับรอบหรือตรวจจับตำแหน่งร่วมกับแม่เหล็ก

## ข้อควรระวัง
- ควรวางแม่เหล็กให้ใกล้กับ Reed Switch พอสมควรจึงจะตรวจจับได้
- หลีกเลี่ยงแรงกระแทก เพราะ Reed Switch เป็นชิ้นส่วนที่มีหลอดแก้วขนาดเล็ก
- ตรวจสอบแรงดันไฟเลี้ยงให้เหมาะกับบอร์ดที่ใช้งาน
- หากอ่านค่าไม่นิ่ง อาจต้องใช้ Pull-up หรือ Pull-down resistor เพิ่มเติม หรือใช้ `INPUT_PULLUP` ใน Arduino

## สรุป
KY-021 เป็นเซนเซอร์แม่เหล็กแบบดิจิทัลที่ใช้งานง่าย ต่อวงจรไม่ซับซ้อน และเหมาะกับงานตรวจจับสถานะเปิด-ปิดหรือการเข้าใกล้ของแม่เหล็ก เหมาะสำหรับผู้เริ่มต้นเรียนรู้ Sensor กับ Arduino
